项目名称：
美股分析 立项背景：
随着金融市场的不断发展，大量的金融数据被生成和积累。这些数据包含了丰富的金融 指标和市场行情，通过对这些数据的分析可以为投资者和决策者提供有价值的信息和洞察。
为了更好地利用这些数据 ，我们计划开发一个使用 Spark RDD 的金融数据分析项目。
该项目旨在提供一个高效、可扩展和易于使用的分析平台，帮助用户从大规模金融 数据中提取有用的信息。

需求分析：
数据获取 ：项目需要从互联网上获取金融数据集 ，可以使用开源的金融数据源 ，如
Yahoo Finance API（https://finance.yahoo.com/）或者其他金融数据 API。
数据集应包含股票价格、指数数据、财务数据等。
数据清洗和转换：获取到的原始数据需要进行清洗和转换 ，确保数据的质量和一致性。 清洗过程可能涉及处理缺失值、异常值和重复数据 ，并将数据转换为适合分析的格式。
数据存储 ：清洗和转换后的数据需要存储到可扩展的数据存储系统中 ，如 Hadoop HDFS。这样可以支持大规模数据的存储和查询。
数据分析：项目需要实现一系列常见的金融数据分析功能，如计算股票收益率、计算指  数移动平均值、计算技术指标（如 RSI、 MACD 等） ，以及基本的统计分析功能（如   均值、方差等） 。这些分析功能需要以可扩展的方式实现 ，以支持大规模数据的处理。 可视化展示：为了更好地理解和呈现分析结果，项目需要提供可视化展示功能。可以使  用开源的可视化库 ，如 Matplotlib 或者 Plotly ，生成各种图表和图形 ，如线图、柱状
图、散点图等。

需求拆解：
1、获取金融数据：
选择合适的金融数据源 ，并使用 API 获取数据。
实现数据下载和存储功能 ，将原始数据保存到本地或者分布式文件系统中。

2、数据清洗和转换：
开发数据清洗模块 ，处理缺失值、异常值和重复数据。
设计数据转换逻辑 ，将数据转换为 RDD 格式 ，并处理数据格式的兼容性。

 3、数据存储：
选择合适的分布式数据存储系统 ，如 HDFS
设计数据存储模块 ，将清洗和转换后的数据存储到选定的数据存储系统中。

4、数据分析：
实现股票收益率计算模块 ，计算每个股票的日收益率。
实现指数移动平均值计算模块 ，计算选定指数的移动平均值。 实现技术指标计算模块 ，如 RSI、 MACD 等。
实现基本统计分析模块 ，如均值、方差等。

5、可视化展示：
选择合适的可视化库 ，如 Matplotlib 或 Plotly。
设计可视化展示模块 ，生成各种金融数据分析图表和图形。
